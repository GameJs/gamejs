fromSurface (surface, threshold)

Creates an image mask from the given Surface. The alpha of each pixel is checked to see if it is greater than the given threshold. If it is greater then that pixel is set as non-colliding.

Parameters

gamejs.Surface surface
Number threshold 0 to 255. defaults to: 255, fully transparent
