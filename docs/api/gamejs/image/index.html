Module gamejs/image

Load images as Surfaces.

Sounds & Images are loaded relative to './'. You can change that, by setting $g.resourceBaseHref in your page before loading the GameJs modules:

<script>
var $g = {
   resourceBaseHref: 'http://example.com/resources/'
}
</script>
<script src="./gamejs-wrapped.js"></script>
....

Functions

  • load(uriOrImage)

load (uriOrImage)

Load image and return it on a Surface.

Preloading

All images must be preloaded:

gamejs.preload(["./images/ship.png", "./images/sunflower.png"]);

before they can be used:

display.blit(gamejs.load('images/ship.png'))

Parameters

String|dom.Image uriOrImage resource uri for image

Returns

gamejs.Surface surface with the image on it.
